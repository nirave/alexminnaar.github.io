How to Multiply a Transposed Distributed RowMatrix in Spark

  • by Alex Minnaar
  • on Mon 05 January 2015

Category: Software Engineering

I was recently building a spark implementation of a topic modelling algorithm which involved a matrix where the rows represented the number of topics and the columns represented the words in the vocabulary. Initially I was storing this matrix locally using a Breeze DenseMatrix. However, when my vocabulary size grew to around 100,000 with a few hundred topics I eventually encountered an out-of-memory error. I had to start using a distributed matrix. Spark's MLlib has three kinds of distributed matrices.

  • RowMatrix - A matrix distributed by row.
  • IndexedRowMatrix - A matrix distributed by an indexed row.
  • CoordinateMatrix - A matrix distributed by entry.
